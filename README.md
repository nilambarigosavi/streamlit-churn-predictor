# 📊 Customer Churn Predictor

This project predicts **customer churn probability** using a machine learning model (Random Forest).

It includes:
- 🔍 A training script (`churn.py`) to process customer data and train the model.
- ⚙️ A Streamlit web app (`app.py`) that takes customer inputs and predicts churn risk.
- 📁 A saved model in the `models/` folder.
